Ones To Watch
for FrontEnd

같은 Notion 기반 뉴스레터 사이트를 열 가지 프레임워크로 각각 정적 빌드한 리팩토링 실험장입니다. 아래에서 변형을 골라 들어가세요.

  • astro 원본 구현 — Content Layer + 정적 빌드, RSS·사이트맵·검색 포함
  • react-router React Router v8 framework mode, ssr:false + prerender
  • tanstack TanStack Start, 정적 prerender + static server functions
  • kudzu HTML-first TSX 실험 프레임워크, getStaticPaths 정적 출력
  • hugo Go 바이너리 SSG — 데이터 파일 + Go 템플릿, 비-Node 툴체인
  • vitepress Vite + Vue 커스텀 테마, 동적 라우트 paths 로더
  • docusaurus React 문서 프레임워크 — 커스텀 플러그인으로 라우트 생성
  • eleventy 11ty v3 — Nunjucks 템플릿 + 전역 데이터 파일, 클라이언트 JS 0
  • next-app Next.js App Router, output:export + generateStaticParams
  • next-pages Next.js Pages Router, output:export + getStaticPaths
